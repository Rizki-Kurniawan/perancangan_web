Penjelasan Tag HTML

<!DOCTYPE html>: Jadi ini menentukan tipe dokumen dan versi HTML yang digunakan. Dalam kasus ini, menunjukkan bahwa dokumen ditulis dalam HTML5.

<html lang="id">: Tag ini mendefinisikan elemen root dari dokumen HTML dan menentukan bahasa dari konten dokumen, dalam hal ini, Bahasa Indonesia.

<head>: Tag ini berisi meta-informasi tentang dokumen HTML, seperti judul, set karakter, dan pengaturan viewport. Ini tidak ditampilkan di halaman web itu sendiri.

<meta charset="UTF-8">: Tag meta ini menentukan pengkodean karakter untuk dokumen, memastikan bahwa karakter diinterpretasikan dengan benar oleh browser. Dalam hal ini, menunjukkan bahwa pengkodean karakter adalah UTF-8, yang mendukung berbagai karakter.

<meta name="viewport" content="width=device-width, initial-scale=1.0">: Tag meta ini mengontrol perilaku viewport pada perangkat mobile. Ini menetapkan lebar viewport ke lebar perangkat dan mengatur tingkat zoom awal ke 1.0, memastikan bahwa halaman web ditampilkan dengan baik pada berbagai ukuran layar.

<title>Penjelasan</title>: Tag ini mendefinisikan judul dokumen HTML, yang ditampilkan di bilah judul atau tab browser. Dalam hal ini, judulnya adalah "Penjelasan".
